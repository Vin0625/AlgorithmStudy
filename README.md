## 스터디 규칙
* 월~금에 3개이상 문제를 풀어 토요일에 올립니다.
* 토요일마다 다음주에 풀 문제를 선정해 공지합니다.
* 언어는 각자의 기호에 맞게 자유롭게 사용합니다.
* 코드만 작성시 주석으로 코드에대한 설명을 추가해주세요.
* 내용설명이 필요할시 md파일로 작성해주세요.
* 매주 풀 문제는 미리 공지합니다.
* 같이 풀고싶은 문제가 있다면 알려주세요.
* 스터디에서 이전에 푼 문제를 올릴시 진행되었던 날자에 해당하는 폴더에 올려주세요.
